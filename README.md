## ü§î What is it?

Eruplan Mobile Client is the mobile application for the Eruplan project. It allows users to manage their family nucleus, add members, and handle emergencies. The application is designed to be a companion to the Eruplan system, providing a mobile interface for key functionalities.

## üöÄ How to try it?

To try the application, you can clone the repository and build it using Android Studio.

```bash
git clone https://github.com/T-R-M-V-spin-off/eruplanmobileclient.git
```

Then, open the project in Android Studio and build it. You can run it on an Android emulator or a physical device.
